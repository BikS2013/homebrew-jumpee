# homebrew-jumpee

Homebrew tap for [Jumpee](https://github.com/BikS2013/jumpee) — a macOS menu bar app for naming and jumping between desktops.

## Installation

```bash
brew tap BikS2013/jumpee /path/to/this/tap
brew install --cask jumpee
```

## After installation

1. Grant Accessibility permissions: **System Settings** > **Privacy & Security** > **Accessibility** > enable **Jumpee**
2. Enable Mission Control shortcuts: **System Settings** > **Keyboard** > **Keyboard Shortcuts** > **Mission Control** > enable **Switch to Desktop 1-9**
3. Press **Cmd+J** to open Jumpee
